This page walks you through installing `stick-to-bottom-svelte` and wiring it up in a Svelte 5 component. By the end you will have a scrollable container that automatically tracks new content and respects user scroll intent, with a spring-physics animation you can tune to match your product.

## Installation

Install the package from npm using your preferred package manager.

<CodeGroup>

```bash npm
npm install stick-to-bottom-svelte
```

```bash pnpm
pnpm install stick-to-bottom-svelte
```

```bash yarn
yarn add stick-to-bottom-svelte
```

```bash bun
bun add stick-to-bottom-svelte
```

</CodeGroup>

## Basic usage

<Steps>
  <Step title="Import StickToBottom">
    Import the class from the package. It is the only export.

    ```typescript
    import { StickToBottom } from 'stick-to-bottom-svelte';
    ```
  </Step>

  <Step title="Create an instance with element getters">
    Instantiate `StickToBottom` inside your component's `<script>`, passing getter functions for your scroll container element and the inner content element. Using getter functions lets the class resolve the elements reactively after they are bound to the DOM.

    ```svelte
    <script lang="ts">
      import { StickToBottom } from 'stick-to-bottom-svelte';

      let scrollElement = $state<HTMLElement>();
      let contentElement = $state<HTMLElement>();

      const stickToBottom = new StickToBottom({
        scrollElement: () => scrollElement,
        contentElement: () => contentElement,
      });
    </script>
    ```
  </Step>

  <Step title="Bind elements in your template">
    Use `bind:this` to connect your DOM elements to the state variables. The outer element must be the scrollable container; the inner element wraps all scrolled content.

    ```svelte
    <div bind:this={scrollElement} class="h-96 overflow-y-auto">
      <div bind:this={contentElement}>
        <!-- content goes here -->
      </div>
    </div>
    ```
  </Step>

  <Step title="Full working example">
    Here is a complete component. New items added to `messages` will trigger an auto-scroll whenever the user is already near the bottom.

    ```svelte
    <script lang="ts">
      import { StickToBottom } from 'stick-to-bottom-svelte';

      let messages = $state([]);
      let scrollElement = $state<HTMLElement>();
      let contentElement = $state<HTMLElement>();

      const stickToBottom = new StickToBottom({
        scrollElement: () => scrollElement,
        contentElement: () => contentElement,
      });

      function addMessage(text: string) {
        messages = [...messages, { text, id: Date.now() }];
      }
    </script>

    <div bind:this={scrollElement} class="h-96 overflow-y-auto">
      <div bind:this={contentElement}>
        {#each messages as message (message.id)}
          <div>{message.text}</div>
        {/each}
      </div>
    </div>
    ```
  </Step>
</Steps>

## Configuration

The constructor accepts spring animation options alongside the required element getters. These three properties control the physics of the scroll animation.

```svelte
<script lang="ts">
  import { StickToBottom } from 'stick-to-bottom-svelte';

  let scrollElement = $state<HTMLElement>();
  let contentElement = $state<HTMLElement>();

  const stickToBottom = new StickToBottom({
    scrollElement: () => scrollElement,
    contentElement: () => contentElement,

    // Spring animation — all optional, defaults shown
    damping: 0.7,    // 0–1: how much oscillation is dampened
    stiffness: 0.05, // 0–1: how quickly the scroll builds speed
    mass: 1.25,      // higher values slow the animation

    // Use 'instant' to skip animation entirely
    resize: 'instant',  // animation used when content resizes
    initial: true,      // set to false to skip the initial scroll
  });
</script>
```

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `damping` | `number` | `0.7` | Amount of damping applied to the spring. `0` = no damping, `1` = full damping. |
| `stiffness` | `number` | `0.05` | How fast the animation accelerates toward the target. |
| `mass` | `number` | `1.25` | Inertial mass. Higher values produce slower, heavier-feeling scrolls. |
| `resize` | `Animation` | spring defaults | Animation used when a content resize triggers a scroll. |
| `initial` | `Animation \| boolean` | `true` | Animation used on first render. Pass `false` to start without scrolling. |

## Reactive state

`StickToBottom` exposes three reactive properties you can read inside `$effect` blocks or bind to your template.

```svelte
<script lang="ts">
  // ...

  $effect(() => {
    if (stickToBottom.isNearBottom) {
      console.log('Auto-scroll is active');
    } else {
      console.log('User has scrolled up — auto-scroll paused');
    }
  });
</script>

<!-- Show a "scroll to bottom" button when the user has scrolled away -->
{#if !stickToBottom.isNearBottom}
  <button onclick={() => stickToBottom.scrollToBottom('instant')}>
    Scroll to bottom
  </button>
{/if}
```

| Property | Type | Description |
| --- | --- | --- |
| `isNearBottom` | `boolean` | `true` when the scroll position is within 70 px of the bottom. Auto-scroll is active while this is `true`. |
| `scrollTop` | `number` | The current `scrollTop` of the scroll element. Reactive — reads from the DOM on each scroll event. |
| `targetScrollTop` | `number` | The maximum scrollable distance (`scrollHeight - 1 - clientHeight`). |
