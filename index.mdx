stick-to-bottom-svelte is a Svelte 5 library that keeps scrollable containers pinned to the bottom as new content arrives — while smartly pausing when the user scrolls up to read. Built on Svelte 5 runes, it gives you reactive state, physics-based spring animations, and full programmatic control with zero runtime dependencies beyond Svelte itself.

## Key features

<CardGroup cols={2}>
  <Card title="Smart scroll detection" icon="eye">
    Automatically pauses when users scroll up to read history, and resumes when they return to the bottom

  <Card title="Spring animations" icon="circle-nodes">

## Get started

<Steps>
  <Step title="Install the package">
    Add stick-to-bottom-svelte to your Svelte 5 project using your preferred package manager.

    <CodeGroup>
    ```bash npm
    npm install stick-to-bottom-svelte
    ```
    ```bash pnpm
    pnpm add stick-to-bottom-svelte
    ```
    ```bash yarn
    yarn add stick-to-bottom-svelte
    ```
    </CodeGroup>
  </Step>
  <Step title="Create a StickToBottom instance">
    Import the class and pass getter functions for your scroll and content elements.

    ```svelte
    <script lang="ts">
      import { StickToBottom } from 'stick-to-bottom-svelte';

      let scrollElement = $state<HTMLElement>();
      let contentElement = $state<HTMLElement>();

      const stickToBottom = new StickToBottom({
        scrollElement: () => scrollElement,
        contentElement: () => contentElement
      });
    </script>
    ```
  </Step>
  <Step title="Bind your elements">
    Attach `bind:this` to your scroll container and content wrapper. That's it — auto-scroll is active.

    ```svelte
    <div bind:this={scrollElement} class="h-96 overflow-y-auto">
      <div bind:this={contentElement}>
        <!-- your content here -->
      </div>
    </div>
    ```
  </Step>
</Steps>
